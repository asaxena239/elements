# Avail Elements

`@rent_avail`

The avail design system is made up of small UI packages that can be mostly imported into projects as needed. This allows updating a package to happen without worrying about breaking changes that might be in development on another components.

---

| ---------------------- | ------------------------------------------------------ |
| @rent_avail/base       | Base, Theme Object                                     |
| @rent_avail/typography | Heading, Text                                          |
| @rent_avail/layout     | Box, Container, Grid, Col, Card, Stack, Flex           |
| @rent_avail/controls   | Button, Checkbox, Radio, Switch                        |
| @rent_avail/input      | Input, Textarea                                        |
| @rent_avail/dialog     | Fullscreen Dialog, Confirmation Dialog, Success Dialog |
| @rent_avail/menu       | Menu                                                   |

## Styled Props

Avail UI elements use [`styled-system`](https://styled-system.com) to help you quickly build custom components with constraint based style props. You can think of style props as utility classes with super powers. Style props inherit from a globally declared theme or use accepted css values. Below is a simple example of using styled props to quickly build a custom [`Card`](/packages/layout#card) component. For more information and to get familiar with all of the possible values, take a look at this [`reference table`](https://styled-system.com/table), to understand how the theme object is built you can review the [`base theme object`](/packages/base)

```jsx
function YourComponent() {
  return (
    <Card my="4rem" bg="blue_100" p="4rem">
      I'm a card with a margin top and bottom of 4rem, a padding of 4rem, and a
      background of our brand color blue_100.
    </Card>
  )
}
```

## Quick Start

First, install the design system and peer dependencies

```bash
yarn add react react-dom styled-components @rent_avail/base @rent_avail/layout
```

Bootstrap your react app and in the root component import the `ThemeProvider` from styled-components and our theme & global styles.

```jsx
import { ThemeProvider } from "styled-components"
import { Base, theme } from "@rent_avail/base"

function App() {
  return (
    <ThemeProvider theme={theme}>
      <Base />
      <Routes />
    </ThemeProvider>
  )
}
```

Now you're ready to start using layout components to quickly build complex UIs.

```jsx
import { Container, Box, Stack, Card } from "@rent_avail/layout"
import { Text, Heading } from "@rent_avail/typography"

function DashboardPage() {
  return (
    <Box as="main" my="4rem">
      <Container as={Stack}>
        <Card bg="blue_100">Item 1</Card>
        <Card bg="blue_300">Item 2</Card>
        <Card bg="blue_500">Item 3</Card>
        <Card bg="blue_700">Item 4</Card>
        <Card bg="blue_900">Item 5</Card>
      </Container>
    </Box>
  )
}
```

## Responsive Styles

Styled system allows us to responsively style our ui components with array props. All styled props accept an array that will map to the breakpoints in the theme spec.

```jsx
<Box width={[1, 1 / 2, 1 / 3]}>
  I'll be 100% on the smallest screens, then 50%, then 33.33%.
</Box>
```

## Theme

You can use the theme properties in any styled component below the `ThemeProvider` component.

```jsx
const Wrapper = styled.div`
  box-shadow: ${({ theme }) => theme.shadows[0]};
  font-size: ${({ theme }) => theme.fontSizes.body};
`
```
